## 关于

HostHarryPotter是一个适用于Among Us的主机端模组。\
模组允许原版玩家游玩到与哈利波特有关的内容，使得原版也能获得更好的体验。\
有关该模组的详细使用文档(安装,更新以及完整的配置说明),请参阅[网站文档]()。
